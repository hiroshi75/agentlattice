# Team Presets

事前定義されたチーム構成テンプレート。企業の管理コンソールで「スタートアップチームを組みたい」等と指示すると、このプリセットに基づいてエージェントを一括雇用できます。

## 使い方

管理コンソール（企業の CLAUDE.md）で:

> 「startup プリセットでチームを構成して」

と指示してください。プリセットの内容を確認の上、各エージェントの個人名をヒアリングしてから順次雇用します。

## 命名規則

エージェント名は `<個人名>-<役割名>` 形式です（例: `jim-ceo`, `alice-marketing`）。
プリセットの `name_template` フィールドの `{name}` 部分にユーザーが指定した個人名が入ります。

## 利用可能なプリセット

| ファイル | プリセット名 | 説明 |
|---|---|---|
| `startup.json` | Startup Team | MVP開発向けの最小チーム（CEO + エンジニア + マーケター + テスター） |
| `dev-team.json` | Development Team | ソフトウェア開発に特化したチーム（PM + フロント + バック + DevOps + QA） |
| `content-team.json` | Content Team | コンテンツマーケティング向けチーム（戦略 + ライター + SEO + SNS） |
